정의 목록
사전과 같이 특정 단어에 대한 정의를 설명하기 위한 용도로 사용할 수 있다.
dl
Description List의 약자이다.
용어(term)와 그에 대한 설명(description)을 리스트 형식으로 정의할 때 사용한다.
dt
Description Term의 약자이다.
용어와 그에 대한 설명을 리스트 형식으로 보여주는 요소에서 용어(term) 부분을 정의할 때 사용한다.
dd
Description Detail 약자이다.
용어와 그에 대한 설명을 리스트 형식으로 보여주는 요소에서 설명(description) 부분을 정의할 때 사용한다.
